# Atomberg Goal Portal Demo Script (3 minutes)

**1. Login as Admin (0:00-0:15)**
- Open the application and navigate to the login screen.
- Enter admin credentials and log in.
- Highlight the role-based dashboard landing.

**2. Show Dashboard with cycle status (0:15-0:30)**
- Walk through the Admin Dashboard.
- Point out the active performance cycle status and company-wide completion rates.
- Mention the real-time data aggregation.

**3. Create a goal and push to team via Shared Goals (0:30-0:50)**
- Navigate to the Shared Goals section.
- Create a new shared goal (e.g., "Reduce B2B TAT").
- Assign it across departments to demonstrate the cascading functionality.

**4. Login as Manager — approve goals, show inline edit (0:50-1:15)**
- Log out and log back in as a Manager.
- Go to the Team section.
- Review pending goals from team members.
- Show inline editing capabilities and approve the goals.

**5. Login as Employee — view goals, submit check-in (1:15-1:35)**
- Log out and log back in as an Employee.
- View the approved goals on the dashboard.
- Submit a new check-in with quantitative progress and notes.

**6. Show Calendar with events (1:35-1:45)**
- Navigate to the Calendar page.
- Show how deadlines, check-in dates, and cycle periods are automatically populated from the database.

**7. Show Innovation Hub and Tech Radar (1:45-2:00)**
- Visit the Innovation Hub. Show the mock ideas, upvotes, and status columns.
- Navigate to the Tech Radar to show the technology assessment framework.

**8. Show AI Chatbot with role-based response (2:00-2:20)**
- Open the Atomberg Intelligence Chat.
- Note the "Offline Mode" badge indicating the fallback mechanism.
- Ask a role-specific question (e.g., "What is my goal completion rate?").
- Show how the response adapts to the logged-in role.

**9. Show Audit Trail and CSV Export (2:20-2:40)**
- Log back in as Admin.
- Go to the Audit Trail section.
- Show the logged actions and demonstrate the CSV export feature.

**10. Show Notification bell with real alerts (2:40-3:00)**
- Click the notification bell in the top right.
- Show the dynamic alerts (e.g., pending check-ins, nearing deadlines).
- Conclude the demo.
